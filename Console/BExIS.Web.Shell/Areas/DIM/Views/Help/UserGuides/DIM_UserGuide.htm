Overview

 

The Data Dissemination Module, available via Setup(Cog button) andExport Metadata , provides a tool to export metadata to a standard compliant XML file. For every metadata structure in the system there is one tab in the tab strip.

The data grid in one tab shows all datasets belonging to the selected metadata structure.

Select a checkbox to mark the datasets you would like to export.

Please click the Export button and wait until the metadata XML file has been created successfully and a download link is available in-line.
